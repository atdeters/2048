# 2048
This is our TUI version of the popular game "2048" by Gabriele Cirulli

Game Keys:
Up: Arrow Up, K, W
Down: Arrow Down, J, S
Left: Arrow Left, A, H
Right: Arrow Right, D, L

Other Keys:
Open Menu: M, 0
Play/Resume: P, 1
Restart: R, 2
